# BMI Calculator - Android Application

A comprehensive and modern Android application to calculate Body Mass Index (BMI), track weight history, and manage multiple user profiles.

## ‚ú® Features

- **Accurate BMI Calculation:** Support for both Metric (KG/CM) and Imperial (Pounds/Feet/Inch) units.
- **Multi-User Support:** Create and manage multiple user profiles within a single app. Switch between users seamlessly.
- **Weight History Tracking:** Automatically saves weight entries to a local database.
- **Visual Weight Graph:** View your weight trends over the last 7 days with a smooth, interactive line chart.
- **Secure Authentication:** User registration, login, and password reset functionality using security questions.
- **Modern UI/UX:** Dark-themed interface with smooth Lottie animations and snappy scrolling for age/weight selection.
